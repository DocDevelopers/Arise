Application Title: Doctor Love
Author: Francisco Castellanos
Language: Java
Platform: Android(4.1)
Rev:3.0

Description:
Doctor Love is a love calculator which takes in two names and generates a random number.
The random number is the the chance of the two names entered to be soul mates.
